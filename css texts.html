Font style css dapat mengatur style dari teks yang ada pada halaman web yang dibuat. terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.
  • text-align → digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right.
